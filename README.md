# UnderwaterRunner

A thrilling underwater running game built with HTML5 Canvas and JavaScript.

## Project Structure

```
UnderwaterRunner/
│
├── assets/
│   ├── images/
│   └── sounds/
│
├── css/
│
├── js/
│   ├── main.js
│   ├── scene.js
│   ├── player.js
│   ├── fish.js
│   ├── boss.js
│   ├── particles.js
│   └── leaderboard.js
│
├── php/
│
├── index.html
├── database.sql
└── README.md
```

## Getting Started

1. Open `index.html` in your web browser
2. Use arrow keys or WASD to control the player
3. Avoid fish enemies and defeat the boss!

## Features

- Player character with smooth movement
- Various enemy types (fish, boss)
- Particle effects
- Leaderboard system
- Sound effects and images

## Technologies Used

- HTML5 Canvas
- JavaScript (ES6+)
- CSS3
- PHP (for backend leaderboard)
- MySQL (for database)
